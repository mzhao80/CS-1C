mzhao80-cs1c-project03/


Brief description of submitted files:

src/stacks/BrowserNavigation.java
    - Simulates a browser's back and forward buttons by recording links that are visited and then keeping a stack of "back" links and a stack of "forward" links.

src/stacks/Navigator.java
    - Contains methods that tell StackList objects what to do for each action available in the browser.

src/stacks/StackList.java
    - Instantiates a StackList of String links for browser navigation. Also defines Node and StackIterator classes for use by StackList.

resources/RUN.txt
    - Console output of BrowserNavigation.java.

resources/links.txt
    - Given example list of links.

resources/popEmptyStackOfLinks.txt
    - Test case of empty StackList of back links.

resources/inputFile01.txt
    - Test case of empty StackList of forward links.

resources/inputFile02.txt
    - Test case of going back when StackList is empty.
