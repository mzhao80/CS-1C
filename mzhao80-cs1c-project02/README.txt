project folder:
mzhao80-cs1c-project02/


Brief description of submitted files:

src/subsetsum/ShoppingBag.java
    - Outputs list of prices of given groceries that are the closest to but do not exceed the user's budget (console input)

src/subsetsum/GroceriesFileReader.java
    - Reads in a file of groceries and extracts their prices

src/subsetsum/SubsetSum.java
    - Methods that solve the subset sum problem and related problems

src/subsetsum/FoothillTunesStore.java
    - Outputs list of songs with durations that are the closest to but do not exceed the target duration (console input)

src/cs1c/TimeConverter.java
    - Times the runtime of other programs

src/cs1c/MillionSongDataSubset.java
    - Parses a JSON file of song metadata and creates SongEntry lists.

src/cs1c/SongEntry.java
    - Object containing metadata of songs from the Million Song Dataset

resources/RUN.txt
    - console outputs of ShoppingBag.java and of FoothillTunesStore.java

resources/groceries.txt
    - Given example list of groceries

resources/inputFile01.txt
    - Other test case #1 list of groceries

resources/inputFile02.txt
    - Other test case #2 list of groceries

resources/music_genre_subset.json
    - Subset of the Million Song Dataset

README.txt
    - description of submitted files
